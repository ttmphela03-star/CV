# CV
This repository contains a few projects I've worked on for Kaggle competitions to showcase evidence behind my technical abilities.
*CustomerChurnRate.html - A visual created in R to showcase to a manager the relationships found behind a great predictor for the field of our concern which is customer churn rate.
*LeavingLowTenureCustomers.html - A follow up visual to show stakeholders or manager reasons behind customers leaving.
*LowTenureCustomers.html - A visual of how many customers are short term customers and other information.
*WhyLongTenureCustomersLeave.html - A visual showing the important reason for long term customers leaving.
*PowerBI_analysis_student_scores - A PowerBI document illustrating a few reasons for the disparity between student test scores.
*kaggle_irrigation_comp_eda.R - A R file of more than 1000 lines of code exploring the relationships beneath variables predicting what irrigation a crop may need in Asia.
*kaggle_irrigation_comp_model.ipynb - Two classification models used to soley focus on the accuracy of predicting the irrigation need of a crop. Python code also includes my understanding of data wrangling and feature engineering.
*neural_network.ipynb - A classifcation neural network built to identify whether or not an F1 driver is likely to stop at the next pit stop. Show casing my understanding of the latest trends within AI and machine learning. 
*MachineLearningDocument.pdf - A digital notebook I've made to update my understanding behind the machine learning models I use. In the hope of one day publishing to help prospective data scientists navigate through the world of machine learning.
